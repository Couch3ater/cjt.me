Education

Rochester Institute of Technology, Rochester NY

In 2014, I earned my B.S. in Media Arts and Technology. Touted to the world as a class of Media Architects, we were made familiar with numerous new mediums like web and mobile, as well as traditional mediums like print.

A regular course load included classes in HTML, digital asset management, print production workflow, basic photography, information architecture, typography, and design. I supplemented these courses with additional classes on database management, networking, computer programming, and some introductory information technology programs, as these courses aligned more with my personal interests.

At Global Printing, my primary role was to create data-driven workflows which would output variable data documents to a number of formats. I regularly found myself using Java, JavaScript, HTML, and a handful of other languages in this position.
